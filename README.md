# 🌐 Portfolio Web - Manuel Polo

Este es mi portafolio web personal, desarrollado con **HTML, CSS y JavaScript**, diseñado para presentar de forma clara y profesional mi perfil, experiencia, habilidades y proyectos.

El enfoque principal del proyecto es ofrecer una experiencia visual moderna, intuitiva y completamente responsive, adaptándose a diferentes dispositivos.

---

## 🚀 Características

- Diseño moderno y responsivo
- Navegación interactiva por secciones
- Presentación profesional del perfil
- Sección “About Me”
- Currículum con experiencia, habilidades y educación
- Portafolio de proyectos y servicios
- Formulario de contacto
- Botón de cambio de idioma

---

## 🛠️ Tecnologías utilizadas

- **HTML5**
- **CSS3**
- **JavaScript**
- **Boxicons**
- **Google Fonts**

---

## 🎯 Objetivo del proyecto

Este portafolio fue creado como una herramienta de presentación profesional, donde puedo mostrar mis conocimientos, experiencia y tecnologías que manejo.

Además, funciona como una plataforma para conectar con oportunidades laborales, colaboraciones y proyectos dentro del área de desarrollo de software.

---

## 💻 Ejecución

Solo necesitas abrir el archivo `index.html` en tu navegador.
